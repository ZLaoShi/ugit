## win系统下安装后的执行
创建父目录ugit，粘贴setup文件到父目录和ugit目录同级，如下
```
Mode                 LastWriteTime         Length Name
----                 -------------         ------ ----
d-----        2024/12/18     14:20                ugit
-a----        2024/12/18     13:27            246 setup.py
```
执行 
```
pip install -e . --no-cache-dir
```
即可安装并使用ugit指令
